# 带传动和链传动

## 传动类型和应用  

### 类型

截面类型分
> 平带 V带 圆带 多楔带

V带的摩擦力  
$$F_f = F_Qf_v$$
$$F_v =\frac{F_Q}{\sin{\frac{\varphi}{2}}}$$
V带传动能力大，且又无接头。传动平稳，应用最广。

带传动优点

1. 适用于中心距较大的传动

2. 带具有良好的挠性，可缓和冲击，吸收振动

3. 过载时带与带轮间产生打滑，可防止损坏其他零件

4. 结构简单，成本低廉。

缺点  

1. 传动的外廓尺寸较大

2. 需要张紧装置

3. 由于带的滑动，不能保证固定不变的传动比

4. 带的寿命短

5. 传动效率较低。
